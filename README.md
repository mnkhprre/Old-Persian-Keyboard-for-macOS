# Old Persian Keyboard for macOS 
 
this layer has been made by Mustafa Bilgin 
developed on Ukulele keyboard editor 

how to install ?
copy the .keylayout file to Macintosh HD/Library/Keyboard Layout directory

		Default=-a layer 
		Shift=-i layer 
 		Opt.=θ/š,log. layer
